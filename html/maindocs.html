USERS API

Create user

Use the Create user API to create a new user

Request type :

POST

Query Params

userData

firstName, lastName, emailId, password

format

JSON

Return

None

Get users

Use this API to get the user data

Request type :

GET

Query Params

id

all --> To get all users data, other number only for that particular user.

format

Integer

Return

JSON

Get users by project

Use this API to get the user data by project id

Request type :

GET

Query Params

id

Only project id number

format

Integer

Return

JSON

JSON

Create user example

Request: GET

FormData :

{


"firstName":"Vigneshwaran"


"lastName":"P"


"emailId":"dummy.email@proapp.com"


"password":"dummy@123"


}

URL: http://ipaddress:8787/ProApp/user/create

Response: Success

Get users example

Request: GET

Param : 2

URL: http://ipaddress:8787/ProApp/user/getusers?id=2

Response:

{


"uid":1,


"userName":"Bharath"


}
